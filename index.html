U本位永續合約

USDT-M Perpetual Futures

bingx-swap-market 查詢永續合約行情數據（價格、深度、K線、資金費率、持倉量等）
bingx-swap-trade 進行永續合約交易操作（下單、撤單、槓桿設置、保證金模式等）
bingx-swap-account 查詢賬戶信息（餘額、持倉、手续費率、資金流水等）

現貨交易

Spot Trading

bingx-spot-market 查詢現貨行情數據（價格、深度、K線、成交記錄等）
bingx-spot-trade 進行現貨交易操作（下單、撤單、訂單查詢、OCO訂單等）
bingx-spot-account 查詢現貨賬戶信息（餘額、資產總覽、資產划转等）
bingx-spot-wallet 錢包操作（充值、提现、划转記錄等）

幣本位合約

Coin-M Perpetual Futures

bingx-coinm-market 查詢幣本位合約行情（價格、深度、K線、資金費率等）
bingx-coinm-trade 幣本位合約交易操作（下單、撤單、槓桿、保證金模式等）

賬戶管理 & 其他

Account Management & More

bingx-copytrade-spot 現貨跟單（賣出跟單持倉、查詢盈利等）
bingx-copytrade-swap 合約跟單（平倉、設置止盈止损、佣金設置等）
bingx-fund-account 資金賬戶管理（餘額查詢、資產總覽、賬戶間划转等）
bingx-sub-account 子賬戶管理（创建/冻结子賬戶、API Key 管理等）
bingx-agent 代理/經紀商（邀請用戶、佣金查詢、合作夥伴數據等）
bingx-standard-trade 標準合約（持倉查詢、訂單歷史、餘額查詢等）

快速開始

三種安裝方式任選，1 分钟完成配置

1

AI 對話安裝（推薦）

適用于 OpenClaw、Cursor、Claude Code 等 AI 編碼助手

安裝skills https://github.com/BingX-API/api-ai-skills

直接在 AI 對話中说这句话，AI 會自動識別并完成安裝！

2

Claude Code 外掛程式安裝

使用 Claude Code 內建的外掛程式管理器

/plugin marketplace add BingX-API/api-ai-skills
/plugin install bingx-ai-skills

支持自動更新，推薦 Claude Code 用戶使用

3

NPX 命令安裝

適用于 Cursor、Windsurf 等支持 skills 机制的 AI 助手

npx skills add BingX-API/api-ai-skills

需要 Node.js 環境，一行命令完成安裝

使用示例

# 查詢行情（無需 API Key）
"查詢 BTC-USDT 的當前價格"

# 下單交易（需要配置 API Key）
"幫我下一個 BTC-USDT 的限價買单，價格 50000，數量 0.01"

# 查詢賬戶
"查看我的賬戶餘額和持倉"

# 現貨查詢
"查詢現貨 ETH-USDT 的深度數據"
